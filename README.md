# WebDev 🚀

A dedicated repository tracking my progress through the **STARTAPPSS Full Stack Roadmap**. This workspace contains my daily projects, code experiments, and foundational concepts as I build my skills from front-end styling to interactive applications.

## 🛠️ Tech Stack
* **HTML5:** Semantic structuring, forms, and accessibility.
* **CSS3:** Flexbox, CSS Grid, media queries, and modern styling.
* **JavaScript:** DOM manipulation, event handling, and closures.

## 📅 Roadmap & Progress

### Week 1: Front-End Foundations & Interactions

* **Day 1: Personal Portfolio & Styling**
  * **Concepts:** Semantic HTML (`<nav>`, `<section>`, `<article>`), Accessibility, Flexbox, Custom Fonts.
  * **Project:** Structured and styled a personal portfolio page with About, Projects, and Contact sections. Implemented a brand color palette, external Google Fonts, and smooth hover effects/transitions.

* **Day 2: Forms & Validation**
  * **Concepts:** HTML5 Input types, native validation, accessibility (`fieldset`, `legend`).
  * **Project:** Built a robust contact form with required fields (name, email, message), custom error handling, and specific pattern matching for robust user input validation.

* **Day 3: Responsive Layouts**
  * **Concepts:** CSS Grid, Flexbox, Media Queries, Mobile-first design.
  * **Project:** Developed a responsive 3-column web layout (Header, Main Content, Sidebar) that smoothly adapts across different device breakpoints.

* **Day 4: High-Fidelity Page Clone**
  * **Concepts:** Translating a design mockup into clean code, visual aesthetics.
  * **Project:** Recreated a beautiful, complex UI layout from a Behance design reference. *Note: Utilizing high-quality, cinematic photography assets for the image placeholders really elevates this layout and gives it a premium feel.*

* **Day 5: JS Basics & The DOM**
  * **Concepts:** JavaScript variables, functions, event listeners, closures, and state management.
  * **Project:** Built an interactive **Counter Application**. Features include increment/decrement controls, boundary limits (max/min), and separate functions handling the logic and DOM rendering to keep the code modular.
